# New Gobo Eiffel Terminal

The command *New Gobo Eiffel Terminal* opens a pre-configured terminal with
*Gobo Eiffel*'s environment set up, so you can run commands directly.

![Integrated terminal](../images/new_gobo_eiffel_terminal.gif)

This command is available in the *Command Palette*. Open the *Command Palette*
with `Ctrl+Shift+P` (or `Cmd+Shift+P` on MacOS) and type `Gobo Eiffel` to see
available commands for this extension.
  
The same command is also available from the right-click contextual menus for
Eiffel files (`.e`) or ECF files (`.ecf`) in both the *Editor* and *Explorer*
panels.
